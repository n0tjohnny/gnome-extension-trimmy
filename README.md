# Trimmy

> **Note**: This is a fork of the original [Trimmer extension](https://github.com/hedgieinsocks/gnome-extension-trimmer) with GNOME 49 compatibility.

## Automatically trim the text in your clipboard

This extension automatically trims whitespace (and optional custom strings) from text when it's copied to the clipboard.

## Features

- Trim whitespace from: both sides, left side only, or right side only
- Optionally trim custom strings from the beginning/end
